# Constraint-Driven Divergence Model (CDDM)

A conceptual framework for severe depression and recursive cognitive-emotional dynamics.

---

## Overview

The Constraint-Driven Divergence Model (CDDM) proposes that severe depression may emerge from recursive cognitive-emotional loops formed under prolonged internal constraint, unresolved affective signaling, and limited adaptive integration capacity.

This repository contains:

- Academic preprint version
- Presentation-style dark version
- Conceptual figure(s)
- Supporting materials related to the CDDM framework
